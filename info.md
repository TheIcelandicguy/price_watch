# Price Watch

Track product prices from any e-commerce site. Paste a URL, get sensors. Powered by Claude AI for universal extraction — no per-retailer parsers needed.

## Highlights

- Universal: works on any product page out of the box
- Smart cost controls keep API spend at $0.50–$2/month for typical use
- Per-product device with price, lowest, highest, target diff, in-stock sensors
- Native HA events for drops, target hits, new lows, restocks
- Optional custom parsers (CSS / regex / JSONPath) for zero-cost tracking on specific sites

Requires an [Anthropic API key](https://console.anthropic.com).
